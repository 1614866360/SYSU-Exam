# SYSU-Exam
本仓库用于整理SYSU期末考试的往年试卷、复习资料，为同学们期末考试助力。

建议使用[Octotree](1)插件进行浏览。

# 贡献文件
欢迎分享你所得到的考卷、复习资料！分享前，请确定以下几个事项：

1.  分享的文件不与仓库中现有的重复
2.  分享的文件是SYSU的考卷、或对SYSU该门课程很有帮助
3.  分享的文件适合公开

## 文件组织形式
视具体情况，选择使用：
*   /科目/试卷文件
*   /科目/专业/试卷文件

## 分享方式
1.  拉取本仓库后，添加你的文件，发起github的pull request
2.  邮箱分享（暂未开通）
